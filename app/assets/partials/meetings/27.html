Programowanie Java Portlets (JSR 168/286/301)

Termin: 18.03.2008 godz. 18:00

Lokalizacja: MIMUW, sala 5440

Materiały
  • dyskusja

O prezentacji:

Celem spotkania jest przedstawienie technologii Java Portlet. Wszyscy którzy nie mieli okazji zetknąć się z tą technologią będą mogli dowiedzieć się m.in.:

  • Portlet i jego cykl rozwojowy w kontenerze portletów
  • Java Portlet 1.0 API (JSR 168)
  • Nowości Java Portlet 2.0 (JSR 286)
  • Różnica między kontenerem portletów a portalem oraz dostępne implementacje wspierające Java Portlet 2.0
  • Zaawansowane szkielety webowe do tworzenia portletów
  • JSR 301 - Portlet Bridge Specification for JavaServer Faces - dostępne implementacje oraz przedstawienie mechanizmu funkcjonowania JSF w środowisku kontenera portletów li Portlety w JSF/JBoss Seam - działanie i potencjalne problemy

Finalna wersja specyfikacji Java Portlet 2.0 (JSR 286) została pozytywnie przegłosowana w JCP 3. marca i czeka na ostateczną publikację!
