Technologia Portali Java na przykładzie JBoss Portal

Bolesław Dawidowicz

Termin: 19.02.2008 godz. 18:00

Lokalizacja: MIMUW, sala 5440

Materiały
  • dyskusja

O prezentacji:

Celem prezentacji jest przedstawienie istoty technologii Java Portlet (JSR 168 oraz 286) oraz jej zastosowań. Uczestnicy dowiedzą się:

  • Czym jest Portlet i jego różnicach w porównaniu do Java Servlet?
  • Jakie korzyści daje zastosowanie portalu i w jakich sytuacjach warto rozważyć jego wdrożenie?
  • Co przynoszą technologie związane z portalami takie jak np. WSRP, SSO, Web Clipping?
  • Dlaczego warto wypróbować implementacje otwarte, a w tym najciekawszy z ciekawych projekt JBoss Portal?
  • W jaki sposób można tworzyć portlety przy użyciu bardziej zaawansowanych szkieletów programistycznych?
  • Czy można integrować portlety z JSF i JBoss Seam?

O prelegencie:

Gdyby nie zapoczątkowana na studiach miłość do Javy i praca z technologią portali zapewne skończyłby jako sysadmin, przez co żywi duży sentyment do systemów operacyjnych... Interesuje się rozwiązaniami z kręgu Identity Management. Aktywny uczestnik Warszawa JUG, który nie daje zapomnieć o istnieniu JBoss Portal.
