O prezentacji:

Podczas prezentacji Filip opowie o Direct Web Remoting (DWR), czyli bibliotece javowej, która umożliwia dynamiczne wywoływanie metod Javy (po stronie serwera) z JavaScriptu (z przeglądarki) wraz z właściwą serializacją przesyłanych obiektów. Podczas prezentacji zobaczymy jak skonfigurować DWR jako serwlet oraz w kontekście Spring Framework, a także wtyczkę DWR do Grails. Zobaczymy również podstawowe funkcje JavaScript API oraz kilka przykładów, co możemy osiągnąć dzięki obiektom DWR. Prezentacja zostanie okraszona fragmentami kodu.

O prelegencie:

Filip Matuszewski jest studentem Wydziału Elektroniki Politechniki Warszawskiej. Od 3 lat pasjonuje się programowaniem. Miłośnik Javy, Spring Framework, Grails oraz dynamicznego wytwarzania aplikacji.
